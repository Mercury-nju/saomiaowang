# 合同扫描王

快速读懂合同 & 识别潜在风险

## 功能

- 📷 拍照扫描合同
- 📄 支持PDF和图片导入
- 🔍 OCR文字识别
- 🤖 AI智能分析关键条款
- ⚠️ 风险识别和提示
- 💬 合同问答
- 📊 报告导出

## 技术栈

- SwiftUI
- Vision (OCR)
- 通义大模型 API
